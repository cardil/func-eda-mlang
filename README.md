# Multi-Language EDA Library for OpenShift Serverless

Experiments in providing a multi-language Event-Driven Architecture (EDA) library for OpenShift Serverless.

## Overview

This repository contains experimental work for developing EDA capabilities across multiple programming languages, designed to work with OpenShift Serverless.
